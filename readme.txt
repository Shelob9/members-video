Create members only videos.

== Description ==
Restrict streaming videos to members only.


* Create a video membership site -- works seamlessly with [Members for Caldera Forms](https://calderawp.com/downloads/members-caldera-forms/)
* Choose your own way to restrict access. See FAQ.
* Sell videos with [Easy Digital Downloads](https://easydigitaldownloads.com/) (Pro Feature)


== Installation ==
Install from WordPress.org or manually.

== Frequently Asked Questions ==
= How do I add videos to a post? =
1. Go to post editor.
1. Open the "Members Video" metabox.
1. Add one or more videos
1. Save post

= How do I make videos show up on a post? =
You have several options:
* The DIY option: Use cwp_mv_show_videos filter to determine access.
* Use [Members for Caldera Forms](https://calderawp.com/downloads/members-caldera-forms/) to show videos to members of a group.
* Sell videos with [Easy Digital Downloads](https://easydigitaldownloads.com/) -- This is a pro add-on coming very soon.
